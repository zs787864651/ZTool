
## 主要技术 ##

- Kotlin
- MVP架构
- Retrofit + Okhttp
- RxJava + RxAndroid
- EventBus
- Glide
- Dagger2
- Material Design
- Logger
- Butterknife
- BaseRecyclerViewAdapterHelper
